Установка гита:
-проверяем систему:
	"Панель управления/Система и безопасность/Система" (смотрим систему 32 или 64)
-скачиваем подходящий установщик
	"с сайта git-scm.com\Downloads"
-запускаем установщик
-при установке выбираем нужный текстовый редактор (не VIM)
-помещаем каталог, где установлена программа git.exe в системную переменную path:
	"Панель управления/Система и безопасность/Система"
		"Дополнительные параметры системы"
			"Переменные среды"
				"Path" (или системная перемнная, или переменная пользователя)
- проверяем конфигурацию гита в файле gitconfig (находится в каталоге etc)


-проверяем конфигурацию для пользователя в файде .gitconfig (в каталоге пользователя)
	должны быть заполнены поля email и name в поле [user]
	если нет, то заполняем командами:
	$ git config user.name <имя пользователя>
	$ git config user.email <почта пользователя>
- создаем рабочий каталог (лучше это делать внутри каталога "Документы")
-заходим в Far рабочий каталог и подаем команду "git --version"
	должна появится версия гита
